Overview

Infernal ("INFERence of RNA ALignment") is for searching DNA sequence databases for RNA structure and sequence similarities. It is an implementation of a special case of profile stochastic context-free grammars called covariance models (CMs). A CM is like a sequence profile, but it scores a combination of sequence consensus and RNA secondary structure consensus, so in many cases, it is more capable of identifying RNA homologs that conserve their secondary structure more than their primary sequence.

The latest release of Infernal is 1.0rc4 [24 Nov 2008]. This is the fourth release candidate for Infernal 1.0 -- and (we believe) the last.

Further reference:

  • Covariance models were originally described in (Eddy and Durbin, 1994). They were first implemented in the COVE software package.
  • The tRNA identification program tRNAscan-SE is based on a tRNA CM and COVE (Lowe and Eddy, 1997).
  • A chapter of Biological Sequence Analysis (Durbin et al., 1998) describes CMs in more detail.
  • The memory-efficient divide-and-conquer CYK alignment algorithm is described in (Eddy, 2002). A new implementation, Infernal, replaced COVE.
  • The RSEARCH software for searching with single RNA structures (instead of with profiles built from multiple alignments) is based on Infernal's alignment engines (Klein and Eddy, 2003). RSEARCH also included MPI parallelization and E-value calculations, features that were merged back into Infernal in Infernal 1.0.
  • The Rfam database of RNA families is based on the Infernal software (Griffiths-Jones, 2003; Griffiths-Jones, 2005) .
  • The QDB query-dependent banded dynamic programming algorithm used to increase search speed is described in (Nawrocki and Eddy, 2007), along with mixture Dirichlet priors and entropy weighting.
